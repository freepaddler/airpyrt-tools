## Change Airport country settings

+ set country (USA)
`acp -q syRe=0x00000000`
+ set [SKU](https://en.wikipedia.org/wiki/Stock_keeping_unit) (FCC)
`acp static apple-sku=FCC`
+ `reboot`

#### SKU list

+ FCC
+ JAPAN
+ ETSI
+ APAC
+ ROW
+ FCC-DFS
+ KOREA
+ RUSSIA
+ SAM
